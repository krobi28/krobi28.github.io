Purpose

The purpose of this weather site is to display, upon user request, many different attributes of the weather. Including current temperature, daily forcast, weekly forcast, current and predicted future wind speed, and wind chill factor. The user will also refer to this website to answer questions about will it rain/snow today, and when, as well as if it will be sunny or partly cloudy. The developers of this website strive to keep you updated on all your weather needs.

Logo


This logo was created by Karen Robison using two different icons found on iconfinder.com. The sun icon indicates this is a website for weather purposes. The thumbs up icon is a universal sybol for "like". The slash through the sun is the universal symbol for stop, no, or not. When all these elements are displayed together, there is a good picture representation of "Weather You Like it or Not"
